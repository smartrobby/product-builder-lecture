🐾 동물상 테스트란?

동물상 테스트는 사람의 얼굴 생김새가 어떤 동물과 닮았는지를 분석하는 재미있는 AI 테스트입니다. Google의 Teachable Machine 기술을 활용해 사진 속 얼굴을 강아지상 또는 고양이상으로 분류합니다.

📸 테스트 방법

  1. 얼굴이 잘 보이는 정면 사진을 준비하세요.
  2. 사진 업로드 영역을 클릭하거나 사진을 드래그해서 올려주세요.
  3. AI가 자동으로 분석하여 결과와 각 동물상 확률을 보여줍니다.
  4. 다른 사진으로 다시 테스트하려면 '다른 사진 분석하기' 버튼을 누르세요.

🐶 강아지상 vs 🐱 고양이상

  • 강아지상: 둥글고 친근한 눈매, 도톰한 입술, 온화하고 밝은 인상이 특징입니다. 사람들에게 호감을 주고 친근감을 느끼게 하는 얼굴형입니다.
  • 고양이상: 올라간 눈꼬리, 갸름한 얼굴형, 도도하고 세련된 분위기가 특징입니다. 신비롭고 매력적인 인상을 줍니다.

🤖 AI 분석 기술

본 서비스는 Google Teachable Machine으로 학습된 이미지 분류 모델을 사용합니다. TensorFlow.js를 기반으로 브라우저에서 직접 실행되며, 업로드된 사진은 서버에 저장되지 않습니다.
