🎰 로또 번호 생성기란?

매주 토요일 추첨되는 대한민국 로또(Lotto 6/45)는 1부터 45까지의 숫자 중 6개를 선택하는 복권입니다. 본 서비스는 검증된 무작위(Fisher-Yates) 알고리즘을 활용해 완전히 공정한 번호 조합 5세트를 즉시 생성해드립니다.

📋 사용 방법

  1. 번호 생성 버튼을 클릭하면 A~E 5세트의 번호가 자동으로 생성됩니다.
  2. 각 세트는 정렬된 6개의 일반 번호와 1개의 보너스 번호(+)로 구성됩니다.
  3. 마음에 드는 번호 조합이 나오면 이미지 저장 버튼으로 저장하세요.
  4. 저장된 이미지를 로또 판매점에서 직접 보여주거나 수동으로 입력하세요.

🎨 번호 색상 안내

  • 🟡 1~10번: 노란색 — 1등 당첨 빈도가 가장 고른 구간
  • 🔵 11~20번: 파란색 — 통계적으로 가장 자주 출현하는 구간
  • 🔴 21~30번: 빨간색
  • ⚫ 31~40번: 회색
  • 🟢 41~45번: 초록색

💡 알아두면 좋은 점

로또는 완전한 무작위 추첨이므로 어떤 번호 조합도 당첨 확률이 동일합니다(1/8,145,060). 본 생성기는 매 클릭마다 독립적으로 새로운 무작위 번호를 생성하며, 이전 결과와 전혀 관계없습니다.
